This is the very first Post on the Blog. It is authored as an MDX file under
`content/posts/` and rendered statically at build time, with zero runtime
JavaScript for the prose itself.

The point of this slice is narrow: prove that a single authored MDX body can
travel from `content/posts/*.mdx`, through the build-time compile pipeline, and
out as rendered HTML on a statically generated `/blog/[slug]` detail page.
