This site is a statically-generated portfolio and blog built on the seam
pattern: domain data, a presentation seam, and components each stay in their
own layer.

The Projects tab is the newest addition — a tab strip and summary view backed
by the same `buildProjectSet` slug-validation gate used everywhere else in the
codebase, plus this Brief route rendering each Project's longer-form MDX body
through the existing `mdxPresentation` seam shared with the Blog.

See [an external reference](https://example.com/projects-tab) for background.
